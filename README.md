2022-1-JAVA2
============
도서 관리 프로그램
------------------
* JAVA와 JSP, DB를 이용한 도서 관리 프로그램

선정 동기
---------
* 여러 주제 중 JAVA와 JSP, DB를 가장 확실하게 사용 할 수 있을 것 같아 선정

주요 기능
---------
* 회원가입, 로그인, 아이디&비밀번호 찾기
* 책 등록&삭제, 대여&반납, 등록된 책 리스트 열람

개발 환경
---------
* 운영체제 : Windows 10
* 개발 언어 : JAVA17
* 서버 : Tomcat 9
* 개발 환경 : Eclipse
